# Multi-Agent Orchestration & Automation

Skill blueprints (prompts, instructions, workflows, operating manuals) that
teach execution models how to coordinate agents and automate verification.

| Skill | Purpose |
|-------|---------|
| [manager-agent-orchestration](manager-agent-orchestration/SKILL.md) | Act as a manager agent: spin up parallel Bug/Performance/Security sub-agent teams with non-overlapping scopes and synthesize their markdown reports into one technical decision report. |
| [async-background-loops](async-background-loops/SKILL.md) | Manage long-running background loops and asynchronous tasks with non-blocking, periodic check-ins instead of freezing on a single process. |
| [frontend-screenshot-pipeline](frontend-screenshot-pipeline/SKILL.md) | Drive sandboxed headless browsers across responsive viewports, inject DOM interaction triggers (menus, hover, focus), and save categorized, timestamp-sorted screenshots. |
| [rubric-based-evaluation](rubric-based-evaluation/SKILL.md) | Objectively score outputs or code variants against a frozen, bias-mitigating multi-point rubric with mandatory concrete citations. |
